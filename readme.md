## Run Locally

Clone the project

```bash
  git clone https://github.com/Massacre-Saint/WebGl-Galaxy-Generator.git
```

Go to the project directory

```bash
  cd WebGl-Galaxy-Generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
